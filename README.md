# Json Position Tool
一个简单的Json坐标读取工具，使用C#编写，支持.NET 8.0。

# 功能：
- **读取Json文件并解析坐标数据。
- **支持多种坐标格式（如经纬度、UTM等）。
- **提供命令行界面，方便用户输入文件路径和选择坐标格式。

# 使用方法：
- ** 克隆或下载此项目。
- ** 使用Visual Studio或命令行工具打开项目。
- **编译并运行程序。
- ** 选择当前目录下的Json文件会自动加载。

# 示例Json格式：
    {
        "Coordinate": [
            {
                "name": "坐标1",
                "x": 695.03,
                "y": 618.11,
                "z": 333.24
            },
            {
                "name": "坐标2",
                "x": 690.08,
                "y": 540.29,
                "z": 327.46
            }
        ]
    }
